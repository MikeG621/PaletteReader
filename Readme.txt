Palette Reader
===============================================================================
Author: Michael Gaisser (mjgaisser@gmail.com)
Version 1.0
Date: 2020.mm.dd
===============================================================================

This editor reads the PLTT resource types found in TIE95 LFD files.

At this time there is no ability to modify the values, this was initially
created to act as an aide in creating images that match the existing palletes
and determining which palettes control the image.

===================
VERSION HISTORY

v1.0 - dd mmm 2020
 - Release

===================
INSTRUCTIONS

- Open the LFD you wish to search using the 'Open' button. The full path will be
displayed next to it. 
- Select the PLTT you wish to view in the list on the left.
- The colors that are included will be shown on the right. Hovering your mouse
over a color will display the RGB values and the index it occupies.
